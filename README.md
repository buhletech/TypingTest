A simple desktop typing test that was built with JavaFx that measures typing speed and accuracy.

- Pre-defined sentence selection from a word bank
- Character color feedback as you type
  - Dark gray: Correct
  - Red: Incorrect
  - Light Gray: Normal text color
Live stopwatch that starts when you begin typing

-- How to Run --
1. Clone the repository
   git clone https://github.com/yourusername/yourrepo.git

2. Open the project in your IDE (IntelliJ recommended)

3. Make sure JavaFX is configured in your project SDK

4. Run TypingTestApplication.java

-- Built With --
- Java
- JavaFX
